Q.16 What are the advantages and disadvantages of using external style sheets?

Ans:-

There are several advantages to using an external style sheet :-

  • An external style sheet allows you to separate the content of your website (HTML) from the presentation (CSS).
  • Reusable styles: An external style sheet can be reused across multiple pages and websites
  • saving time and making it easier to maintain
  • An external style sheet is only loaded once, even if it is used on multiple pages. This can improve the performance of your website

There are also some disadvantages to using an external style sheet :-

  • Limited control: With an external style sheet, you have less control over the specific elements on a page, as the styles are applied globally to all elements that use the same class or ID.
  • It can be harder to override the styles in an external style sheet, as they are applied globally. To override a style, you need to use more specific selectors or use the !important declaration, which can make your style sheet more complex and difficult to maintain.

Q.17 What is the meaning of the CSS selector

Ans:-

CSS selectors are used to select the content you want to style. Selectors are the part of CSS rule set. CSS selectors select HTML elements according to its id, class, type, attribute etc. There are several different types of selectors in CSS.

  • Basic selector
  • Combinator Selector
  • Group selector
  • Psuedo selector

Q.19 What is the rule set?

Ans:-

A CSS ruleset consists of an element selector and a properties declaration block.

  • A selector selects HTML elements. In other words, developers use CSS selectors in a stylesheet to select the HTML elements they which to style.
  • A CSS declaration consists of a CSS property name and value separated by a colon (:).
